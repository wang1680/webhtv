# 网盘选集标题精简规则

## 调研来源

- 真实网盘/资源站搜索样本：夸克、阿里云盘、百度网盘、115、SouPan/资料库类索引、短剧合集页面。
- 开源文件名解析器参考：`guessit-io/guessit`、`erengy/anitomy`。
- 既有日志样本：国漫年番、短剧全集、更新至、4K/中字/国语等混合标题。

## 常见命名类型

| 类型 | 样例 | 处理策略 |
| --- | --- | --- |
| 平台/资源站广告 | `百度云网盘夸克下载`、`阿里云盘`、`免费电影下载` | 只在同一组选集形成公共前后缀时压缩，不单独按关键词删除 |
| 合集状态 | `全42集`、`更新至第146集`、`第129-180集` | 保留，不把 `集` 字拆掉 |
| 季/年番/篇章 | `第二季`、`年番3`、`第四季`、`前篇/后篇` | 保留，避免和集数混淆 |
| 集数定位 | `第01集`、`01集`、`EP01`、`E01`、`S01E02`、`145集` | 必须保留完整 token |
| 版本修订 | `v2`、`修复版`、`导演剪辑版`、`加长版`、`特别篇` | 保留，避免用户选错版本 |
| 画质分辨率 | `4K`、`2160p`、`1080p`、`720p` | 混合时保留；全组选集完全相同时可作为公共后缀压缩 |
| 片源 | `WEB-DL`、`WEBRip`、`BluRay`、`BD`、`HD` | 混合时保留；全组选集完全相同时可压缩 |
| 视频编码 | `HEVC`、`H265`、`H.265`、`H264`、`AV1` | 混合时保留；全组选集完全相同时可压缩 |
| HDR/DV | `HDR`、`HDR10`、`DV`、`Dolby Vision` | 混合时保留；全组选集完全相同时可压缩 |
| 字幕/音轨 | `国语`、`粤语`、`中字`、`中英双字`、`无字` | 混合时保留；全组选集完全相同时可压缩 |
| 发布组/字幕组 | `[TaigaSubs]`、`-CatWEB` | 只按公共前后缀处理，不硬删 |
| 文件属性 | `.mkv`、`.mp4`、`[1.2GB]`、`[1234ABCD]` | 开启后清理末尾扩展名、体积、括号校验 |
| 短剧全集类 | `短剧合集500+部`、`全集完整版` | 保留，除非它是同一组选集公共前后缀 |

## 当前实现边界

- 默认关闭：播放设置弹窗里新增 `精简标题`，用户开启后才生效。
- 不修改 `Episode.name`、`Episode.desc`、`Episode.url`，只设置临时 `displayName`。
- 不在滚动绑定中动态计算；选集列表刷新或列宽计算时一次性预处理。
- 手机端和电视端选集按钮都读取 `Episode.getDisplayName()`。
- 只压缩同一组选集的公共前缀/后缀，并要求切点位于分隔符边界，或切点后完整像集数 token。
- 压缩后仍然很长时，优先展示明确集数 token，例如 `S01E07`、`第07集`、`EP07`、`07集`。
- 对 `仙逆145集`、`仙逆146集` 这类无分隔命名，只允许压缩到 `仙逆`，不会压缩到公共数字 `仙逆14`。
- 如果压缩后出现重复展示名，重复项回退到清理后的原标题。

## 不做的事

- 不按关键词硬删 `4K`、`中字`、`国语` 等信息，因为不同资源混合时这些是用户选择依据。
- 不重写为完整媒体文件名解析器；真实网盘标题来源混杂，完整解析误伤成本高。
- 不影响默认体验；关闭开关后展示完全回到原始标题。
